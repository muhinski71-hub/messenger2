# Десктоп-приложение (Windows)

Приложение — это мессенджер, открытый в отдельном окне браузера (app mode).
Так надёжно работают **микрофон**, **голосовой чат** и **демонстрация экрана**.

## Установка

Дважды кликни по файлу `create_shortcut.ps1` → «Выполнить с помощью PowerShell».
Если не запускается, открой PowerShell в этой папке и выполни:

```powershell
powershell -ExecutionPolicy Bypass -File create_shortcut.ps1
```

После этого на **рабочем столе** появится ярлык **«Мессенджер»**.

## Использование

- Запусти ярлык «Мессенджер» с рабочего стола — откроется отдельное окно.
- Первый раз браузер спросит разрешение на микрофон/демонстрацию экрана — нажми «Разрешить».
- Приложение подключается к `https://messenger-jf87.onrender.com`.

## Сменить адрес сервера

Открой `create_shortcut.ps1`, поменяй строку `$url = "..."`, запусти скрипт заново.
